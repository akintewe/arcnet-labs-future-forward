# Arcnet Labs - Future Forward

## Project Overview

Arcnet Labs is an AI-first engineering lab building intelligent products, automation systems, and next-generation digital experiences. This is the official website for Arcnet Labs.

## Getting Started

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion
- React Router

## Deployment

This project is configured for deployment on Netlify. Changes pushed to the `main` branch will automatically trigger a deployment.
